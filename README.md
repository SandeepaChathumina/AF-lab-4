# SE3040 - Application Frameworks: Lab 04
## Introduction to Node.js and GitHub Flow

### Project Description
This repository contains the practical implementations for Lab 04 of the 3rd Year Software Engineering specialization at SLIIT. The lab focuses on mastering the basics of **Node.js** and professional **Git/GitHub collaboration** workflows.

### Lab Tasks Completed:
* **Initial Setup:** Initialized the project using `npm init` and established the main `app.js` entry point.
* **File System Operations:** Implemented reading from and writing to local files using the `fs` module.
* **Web Server:** Created a basic HTTP server listening on port 8080.
* **HTTP Requests:** Performed an HTTPS GET request to fetch data from an external REST API.
* **Modular Programming:** Developed and utilized custom Node.js module.
* **Asynchronous Programming:** Explored modern JavaScript patterns using both Promises and Async/Await.
* **Git Collaboration:** Practiced branch management, Pull Requests (PRs), and collaborator merging.
